# AbelTomato's Blog

个人博客，记录学习笔记、项目实践与技术思考

在线访问：[https://abeltomato.github.io](https://abeltomato.github.io)

## 技术栈

- [Astro 5](https://astro.build/)：静态站点生成
- [React 19](https://react.dev/)：交互式组件
- [TypeScript](https://www.typescriptlang.org/)：严格类型检查
- [Tailwind CSS 4](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)：样式与组件基础
- Markdown / MDX：博客内容编写，支持数学公式与代码高亮
- GitHub Actions + GitHub Pages：自动构建与部署
